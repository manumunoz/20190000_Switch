The Model: 3 Stages

    • Players are exogenously assigned a group identity.
    • They can endogenously change it.

  • 2. Network formation

    • Players observe the group identity of others.

General


  • • There is a set of players: $N=\{1,2\}$
  • • Each player $i$ is assigned a group identity: $θ_i=\{●, ▲\}$, and
  • • An external attribute (appearance): $φ_i=\{◯, △\}$
  • • Every $ i ∈ N $ knows the group assigned to all other players: $θ=\{θ_1,θ_2,..., θ_n\}$
  • • The set of players by group: $N_{●}=\{\i ∈ N: θ_i = ●\}$ and $N_{▲}=\{\i ∈ N: θ_i = ▲\}$, with cardinality $|N_{●}|,|N_{▲}|$
  • • If $|N_{●}|≠|N_{▲}|$, the largest set is the majority

1. Group identity choice


  • • Endogenous choice of group: $\ov{θ}_i=\{●, ▲\}$
  • • $i$ can stay in the assigned group, $(\ov{θ}_i=θ_i)$, or change it, $(\ov{θ}_i ≠ θ_i)$
  • • The resulting vector of chosen groups is: $\ov{θ}=\{\ov{θ}_1,\ov{θ}_2,..., \ov{θ}_n\}$
  • • Finite set of all vectors: $\ov{Θ}$

2. Network formation


  • • Each player observes $\ov{θ}$
  • • Proposes connections to others: $g_i=(g_{i1},...,g_{ii-1},g_{ii+1},...,g_{in})$ where $g_{ij} ∈ \{0,1\}$
  • • $N_i\{g\}=\{\j ∈ N: g_{ij} = 1\}$ is $i$'s targeted neighbors (outdegree)
  • • Results in the directed network $g=(g_1,g_2,...,g_n)$
  • • If $g_{ij}=g_{ji}=1$ an undirected link $\ov{g}_{ij}$ is formed
  • • The resulting network $\ov{g}$ represents all bilateral connections: $N_i\{\ov{g}\}=\{\j ∈ N: \ov{g}_{ij} = 1\}$
  • • Finite set of all undirected networks: $\ov{G}$

3. Action chosen


  • • All players observe the resulting network (all proposals)
  • • $i$ chooses an action from a binary set $A=\{$p,b$\}$
  • • Group based preference: $● (▲)$ prefers p over b (b over p)
  • • $x_i(\ov{g})=\ov{θ}_i$ when $i$ chooses her preferred action, and $x_i(\ov{g}) ≠ \ov{θ}_i$ otherwise
  • • $i$'s strategy is defined through a total function $x_i:\ov{Θ},\ov{G} → A$

Utility (payoffs)


$$u_i(\ov{θ}_i,g,\ov{g},x)=λ^{\ov{θ}_i}_{x_i}(1+∑_{j ∈ N_i(\ov{g})} I_{\{x_j=x_i\}})- (c(\ov{θ}_i)+ k|N_i(g)|)$$

  • • where $∑_{j ∈ N_i(\ov{g})} I_{\{x_j=x_i\}}$ counts all neighbors choosing the same as $i$
  • • $λ^{\ov{θ}_i}_{x_i}=α$ if $x_i(\ov{g})=\ov{θ}_i$ (chooses her preferred action) and $λ^{\ov{θ}_i}_{x_i}=β$ otherwise: $α>β$
  • • switching cost: $c(\ov{θ}_i) ≥ 0$ if $i$ changed groups $(\ov{θ}_i ≠ θ_i)$, and $0$ otherwise
  • • Cost of proposing a link: $k>0$

Equilibrium charaterization (B.I.)

1. Actions: Fix the network. Players choose their preferred action if the proportion selecting the same is sufficiently large (threshold).
2. Connections: Every two players choosing the same action are connected, every two players connected are choosing the same action (Pairwise stability).

Equilibrium charaterization (B.I.)

1. Actions: Fix the network. Players choose their preferred action if the proportion selecting the same is sufficiently large (threshold).
2. Connections: Every two players choosing the same action are connected, every two players connected are choosing the same action (Pairwise stability).
3. Group choice: All players' group identity align with the chosen action.

Fixed vs. Fluid Group Identities

Effects on equilibria and efficiency

  • Equilibria

    Identity change shrinks the equilibrium set to outcomes where all players choose their preferred action
    *even compared to fixed identities with homogeneous populations.

  • Efficiency gains

    The efficient outcome (complete network with coordination on a single action) leads to larger gains with fluid identities.
